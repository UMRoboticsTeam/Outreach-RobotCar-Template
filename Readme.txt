
Description:
    This Project is meant to be used as a template for the Elegoo Robot Cars that are used by the UMRT Outreach Team.

Purpose:
    The main purpose of this project is for you to use this as a starting point for your own projects. 
    Please clone the repo and then start a new GitHub repo for your own projects.

Source Files provided:
- Main_instructor_copy.ino: Example solution for use by workshop instructors
- Main.ino: Blank template to be used in workshops.

Functions Provided:

Movement:
    - Forward
    - Reverse
    - turn Right
    - turn left
Sensor:
    - Ultrasonic distance measurement
